# Vivian Yang Portfolio

Standalone public portfolio app for Vercel deployment.

## Deploy
1. Push this folder into the `vivianyang` GitHub repo.
2. Import the repo into Vercel.
3. Optionally connect a custom domain.

## Notes
- Keep secrets in Vercel environment variables, never in source.
- VC lead captures are written to a local JSONL log by default and can be upgraded to a CRM or notification workflow later.
